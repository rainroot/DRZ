* Drizzle-vpn [ DZ-VPN ]

DZ-vpn은 openvpn의 프로토콜을 기반으로 멀티스레드를 이용하여 성능을 극대화 하였으며,
openvpn의 프로토콜을 사용하여 openvpn과 호환이 가능하다.

 openvpn <--> dz-vpn 터널링 가능.( 단 TCP 만 가능 )
 
 
 성능 비교.
 
 openvpn 최대 150Mbps
 
 dz-vpn 최대 1.2Gbps
